# Stealthify V2

Stealthify V2 is a lightweight code obfuscator for Python, designed to provide basic protection against casual reverse engineering of your source code.
It applies simple obfuscation techniques to make your code less readable and deter unauthorized access.

# Features:
Basic Name Obfuscation: Renames with randomly generated names to make the code less intuitive.

Junk Code Injection: Adds randomly generated lines of junk code throughout the code to make it harder to analyze and understand.

AST Transformations: Performs transformations on the Abstract Syntax Tree (AST) of the code to modify its structure and confuse potential attackers.

Encoding and Compression: Applies additional layers of Base64 encoding, base85 encoding, marshal encoding, and lzma compression to the obfuscated code to make it harder to understand.

Startup Code: Provides an option to add code that enables the script to run on startup.

Multiple Obfuscation Layers: Allows the user to choose and apply multiple layers of obfuscation to the code, increasing its complexity.

Basic Anti-VM Code: Includes code to detect and bypass Virtual Machine environments.

# Usage:

Ensure that the required modules:
ast, random, base64, os, marshal, zlib, sys, colorama. are installed in your Python environment.

1. Save the script to a file, for example, obfuscator.py.
2. Open a terminal or command prompt and navigate to the directory where obfuscator.py is saved.
3. Run the script by executing the command python obfuscator.py.
4. Follow the prompts to provide the file path of the code you want to obfuscate, the number of obfuscation layers, and other options.
5. Once the process is complete, the obfuscated code will be saved to a new file with the suffix "_obf" added to the original file name.
6. Please note that while obfuscation can provide a basic level of code protection, it does not guarantee absolute security.
7. attackers may still attempt to reverse engineer the code given time and expertise.
